Design: Architecture & Stack

Section 1 of 5 — does this look right so far?

Stack

Frontend

  • ⚙️ Next.js 15 App Router
  • 🎨 Tailwind CSS + shadcn/ui
  • ✍️ Tiptap rich text editor
  • 🌐 next-intl i18n routing

Backend / Infra

  • 🗄️ Supabase Postgres + Auth + Storage
  • 🔒 Row Level Security for permissions
  • 🌍 DeepL API auto-translation
  • 🚀 Vercel hosting + Edge Functions

Request Flow

Browser
→
Vercel Edge
→
Next.js App Router
→
Supabase (DB + Auth)
Public pages (SSR)

Homepage, article, author — rendered server-side for SEO

Auth-gated (RSC)

Dashboard, CMS, profile — server components with session check

API Routes

Likes, bookmarks, translation triggers — lightweight Next.js route handlers

Translation Pipeline

Author publishes (ES or EN)
→
Supabase DB trigger
→
Edge Function calls DeepL
→
Stores both ES + EN in DB

DeepL free tier: 500K chars/month. A typical 1000-word article ≈ 6K chars. ~83 articles/month before any cost. Paid: $5.49/month for 1M chars.
