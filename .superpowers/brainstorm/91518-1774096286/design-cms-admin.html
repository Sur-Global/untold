Design: CMS & Admin

Section 5 of 5 — content creation, moderation, admin dashboard

Content Creation (all authenticated users)

Article editor (Tiptap)

  • Rich text: headings, bold, italic, lists, blockquotes
  • Image upload → Supabase Storage
  • Embed blocks: YouTube, Spotify, Twitter/X
  • Cover image upload
  • Categories + tags picker
  • Excerpt field
  • Save draft / Publish
  • Auto-saves every 30s

Other content types

  • Video — title, embed URL, description, thumbnail, duration, tags
  • Podcast — title, embed URL, episode no., description, cover, duration
  • Knowledge Pill — title, short text, color, optional image, tags
  • Course — admin only · title, price, currency, duration, Stripe product ID

Admin Dashboard panels

Content Management
  • List all content (all types)
  • Filter by type, status, author
  • Feature / unfeature
  • Publish / unpublish / delete
  • Preview any content
User Management
  • List all users
  • Promote user → author
  • Demote / suspend
  • View user's content
  • Review "Become Creator" requests
Analytics
  • Total users, authors, content
  • Most liked content
  • Most bookmarked content
  • New signups over time
  • Content by type breakdown

Translation trigger

When content is published, a Supabase Edge Function fires: translates title, excerpt, and body into all configured locales (ES, PT, FR, DE, DA) via DeepL. Stores results in content_translations with is_auto_translated = true. Admin can manually override any translation. Quechua: manual only — admin submits translation directly.
